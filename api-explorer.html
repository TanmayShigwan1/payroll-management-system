Database Structure

The application uses a PostgreSQL database with the following main tables:

  • employees - Stores employee information with inheritance strategy
  • payrolls - Stores payroll information for each pay period
  • pay_slips - Stores pay slip records generated from payrolls

Docker Services

The application is running in Docker containers:

  • PostgreSQL - Running on port 5432
  • Spring Boot Backend - Running on port 8080
